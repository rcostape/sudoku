O Jogo tem por objetivo concorrer a bolsa de estudo no IESB.

Regras do Jogo
Não está sendo realizado todo o tipo de checagem, tendo em vista que
o objetivo é do desafio é verificar a capacidade dos concorrentes no
desenvolvimento de software.

Logo, entradas com valores diferentes dos previstos, tais como valores
nulos, negativos, letras, entre outros, podem causar erros inesperados.
Além disso, não foi criada mensagem específica para cada erro para não
dar pistas para o jogador. A ideia é apresentar uma mensagem genérica:
"Jogada não permitida!".

Não há interface gráfica, logo as entradas dos dados se dará pela Console.

O tabuleiro já está praticamente preenchido, para que seja possível verificar a implementação de chacagens necessárias:
Linhas;
Colunas;
Grupo;
Não substituição de números inseridos no início do jogo.

Logo, para ter mais opções de jogada, é necessário apagar as células
iniciais previamente preenchidas.

Por fim, é possível substituir número preenchido pelo jogador e chegar na
finalização do jogo.
